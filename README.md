# Ollama Cog Model - Llama3.2:1b

This is an implementation of the Ollama model [Llama3.2:1b](https://ollama.com/library/llama3.2) as a [Cog](https://github.com/replicate/cog) model.

## Development

Follow the [model pushing guide](https://replicate.com/docs/guides/push-a-model) to push your own model to [Replicate](https://replicate.com).
    
## Basic Usage

To run a prediction:

    cog predict -i prompt="tell me a joke"


## Output

    Sure, here's a light one for you:

    Why don't scientists trust atoms?

    Because they make up everything!
